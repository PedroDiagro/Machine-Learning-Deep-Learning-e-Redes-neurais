# 🤖 Machine Learning & Neural Networks Studies

Este repositório reúne experimentos e implementações de algoritmos de Machine Learning e Redes Neurais desenvolvidos em Python ao longo das disciplinas da graduação.

O foco está em compreender o funcionamento dos modelos além do uso de bibliotecas prontas, analisando treinamento, desempenho e comportamento dos algoritmos em diferentes conjuntos de dados.

---

## 🎯 Objetivo

* Entender fundamentos de aprendizado de máquina
* Implementar e avaliar modelos supervisionados
* Analisar comportamento dos algoritmos nos dados
* Explorar conceitos de treinamento e generalização
* Construir base teórica para aplicações em IA

---

## 📂 O que você vai encontrar aqui

* Regressão linear
* K-Nearest Neighbors (KNN)
* Classificação e regressão supervisionada
* Experimentos realizados no Google Colab
* Comparações entre modelos

---

## 🛠 Tecnologias

* Python
* Scikit-learn
* Google Colab
* Numpy / Pandas / Matplotlib

---

## 📈 Em constante evolução

Novos algoritmos e técnicas serão adicionados conforme o avanço das disciplinas, incluindo modelos mais complexos e redes neurais profundas.
